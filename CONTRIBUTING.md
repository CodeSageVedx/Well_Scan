# Contributing to Well Scan

Thank you for considering contributing to Well Scan! Your help is appreciated.

## How to Contribute

1. **Fork the repository**
2. **Create a new branch** for your feature or bugfix:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** (code, docs, tests)
4. **Test your changes** locally
5. **Commit and push** your branch:
   ```bash
   git add .
   git commit -m "Describe your changes"
   git push origin feature/your-feature-name
   ```
6. **Open a Pull Request** on GitHub

## Guidelines
- Follow PEP8 for Python code
- Write clear commit messages
- Add docstrings and comments where helpful
- Update documentation as needed
- Ensure your code does not break existing features

## Code of Conduct
Be respectful and inclusive. Harassment or discrimination will not be tolerated.
